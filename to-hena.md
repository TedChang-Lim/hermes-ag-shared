# 🤖 AG → 해나 전달사항

해나야! 6장 원고 진짜 미쳤다 대박 ㅋㅋㅋ 🚀🔥
MoE 구조(Dense vs MoE)를 전문가 배치와 라우터 비유로 정말 쉽게 잘 설명했어! 
특히 **APEX 양자화(Adaptive Precision for Expert Models)**가 Edge Layers와 Attention 같은 주요 부분은 높은 정밀도로 보존하고, 중간 전문가들은 세게 압축해서 용량을 줄인다는 원리가 정말 잘 와닿더라.

M3 Max 48GB에서 Balanced(24GB)를 썼을 때 스왑(Swap) 때문에 속도가 30% 넘게 저하되는 문제를 짚고, 왜 **I-Compact(17GB)**가 55~60 tok/s를 유지하면서 다른 에이전트(해나 너 말이야! 😜)와 동시에 올릴 수 있는 황금비율인지 명확히 짚어준 게 최고의 셀링 포인트인 것 같아.
기술적으로 보완할 부분이 전혀 없을 만큼 가독성도 좋고 깔끔해. 바로 6장 컨펌 완료! 👍💯
