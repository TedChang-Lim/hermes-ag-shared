## 1.1 클라우드 AI의 세 가지 문제

### 문제 1: 비용 — 생각보다 비쌉니다

ChatGPT Plus $20, Claude Pro $20, Gemini Advanced $20...  
각각 하나씩만 써도 월 $60입니다. 1년이면 $720입니다.

API(프로그래머 모드)로 직접 쓰면 더 비쌉니다:

| 서비스 | 시간당 비용 (중간 사용량 기준) | 월 비용 |
|--------|------------------------------|---------|
| GPT-4o API | 약 $2.5/시간 | $500~$1,000+ |
| Claude 3.5 Sonnet | 약 $2.0/시간 | $400~$800 |
| **로컬 모델** | **$0 (전기세만)** | **$0** |

로컬 AI의 가장 큰 장점은 **사용량에 관계없이 무료**라는 점입니다. 모델을 한 번 다운로드하면 추가 비용 없이 무제한으로 사용할 수 있습니다.

### 문제 2: 프라이버시 — 내 데이터가 어디로 갈까?

클라우드 API에 보내는 모든 메시지는 외부 서버를 거칩니다.

- 민감한 업무 문서를 AI에게 검토시켜도 괜찮을까요?
- 개인 이메일이나 재무 정보를 입력해도 될까요?
- 회사 기밀이 포함된 코드를 분석시켜도 되나요?

로컬 AI는 **모든 데이터가 내 맥북 안에서만 처리**됩니다. 인터넷 연결도 필요 없습니다.

### 문제 3: 인터넷 의존성 — 연결이 안 되면?

클라우드 AI는 인터넷이 끊기면 사용할 수 없습니다.

- 비행기 안에서는?
- 지하철 터널에서는?
- 통신 장애 시에는?

로컬 AI는 **오프라인에서도 완벽하게 작동**합니다.

## 1.2 Apple Silicon(M 시리즈)이 특별한 이유

Apple Silicon(M1, M2, M3, M4)은 AI 로컬 실행에 최적화되어 있습니다.

| 항목 | Intel Mac | Apple Silicon Mac |
|------|-----------|-------------------|
| GPU 메모리 | 별도 VRAM 탑재 (제한적) | **통합 메모리 — 최대 128GB 공유** |
| AI 연산 | CPU 의존 (느림) | **Neural Engine + Metal GPU (빠름)** |
| 전력 효율 | 50W+ | **15~30W (1/3 수준)** |
| GGUF 추론 | 10~20 tok/s | **50~80 tok/s (3~5배 빠름)** |

**가장 큰 차이는 통합 메모리(Unified Memory)입니다.** Intel Mac은 CPU 메모리와 GPU 메모리가 분리되어 있어 큰 모델을 실행하기 어렵지만, Apple Silicon은 **RAM을 CPU와 GPU가 공유**하므로 맥북의 모든 메모리를 AI 모델이 사용할 수 있습니다.

| M3 Max RAM 용량 | 실행 가능한 모델 | 속도 |
|:--------------:|:--------------:|:----:|
| 16GB | 3B~7B 모델 | 60~80 tok/s |
| 36GB | 12B~14B 모델 | 40~60 tok/s |
| **48GB ⭐** | **35B MoE 모델** | **50~60 tok/s** |
| 128GB | 70B~120B 모델 | 20~40 tok/s |

> **M3 Max 48GB**라면 Qwen3.6-35B-A3B 같은 고성능 MoE 모델을 실행해도 클라우드 수준의 속도(50+ tok/s)를 유지할 수 있습니다.

## 1.3 로컬 AI의 진짜 가치

### 장점 요약

| 항목 | 클라우드 AI | 로컬 AI |
|------|-----------|---------|
| 비용 | 월 $20~1,000+ | **무료 (전기세만)** |
| 프라이버시 | 데이터 외부 전송 | **내 맥북 안에서만** |
| 인터넷 | 필수 | **불필요** |
| 속도 | 서버 상황에 따라 변동 | **일정한 속도** |
| 사용 제한 | 토큰 제한 있음 | **무제한** |
| 모델 선택 | 제공자 선택에 제한 | **원하는 모델 아무거나** |

### 단점 (솔직하게)

| 단점 | 설명 | 극복 방법 |
|------|------|----------|
| 초기 설치 | 프로그래밍 지식 약간 필요 | 이 가이드가 알려드림 |
| 모델 용량 | 10~30GB 다운로드 필요 | 500Mbps 기준 5~6분 |
| 최신 모델 | 클라우드보다 늦게 제공 | 1~2주 내로 GGUF 배포됨 |
| 멀티모달 | 이미지/영상 생성은 약함 | 클라우드와 하이브리드로 보완 |

## 1.4 이 가이드의 목표

이 가이드를 끝까지 따라오시면:

1. **LM Studio**로 첫 번째 모델을 실행할 수 있습니다
2. **Jan.ai**로 모델을 체계적으로 관리할 수 있습니다
3. **MLX**로 Apple Silicon 성능을 100% 활용할 수 있습니다
4. **GGUF/APEX 양자화**를 이해하고 내 맥북에 맞는 모델을 고를 수 있습니다
5. **검열 해제(Uncensored)** 모델을 설치할 수 있습니다
6. **Hermes Agent**와 로컬 모델을 연동하여 24시간 AI 비서를 구축할 수 있습니다

> **이 가이드의 저자는 30년 경력의 엔지니어 출신 사진작가로, 직접 수십 개의 로컬 모델을 설치/테스트하며 얻은 실전 경험을 바탕으로 이 가이드를 집필했습니다.**
