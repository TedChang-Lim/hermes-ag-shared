#### 1.1 AI API 가격, 왜 문제인가?

ChatGPT Plus는 월 $20, Claude Pro는 월 $20, Gemini Pro는 월 $20...

1인 기업가가 AI 비서를 제대로 쓰려면 최소 2~3개의 구독이 필요하다. 월 $60~100. 1년이면 $720~1,200.

문제는 이것뿐만이 아니다. API(프로그래머용)로 직접 쓰면 더 비싸다.

| 서비스 | 입력 가격 (1M tokens) | 출력 가격 (1M tokens) |
|--------|---------------------|---------------------|
| GPT-4o | $5.00 | $15.00 |
| Claude 3.5 Sonnet | $3.00 | $15.00 |
| Gemini 2.5 Pro | $1.25 | $5.00 |

하루 1,000만 토큰만 써도 GPT-4o 기준으로 하루에 $58.75다. 한 달이면 $1,700을 훌쩍 넘긴다.

이게 'AI 비서를 못 쓰는' 가장 큰 이유다.

#### 1.2 DeepSeek V4, 가격 혁명을 일으키다

2025년 12월, 중국 DeepSeek이 V4 시리즈를 발표하면서 시장이 흔들렸다. 단순히 성능이 좋아서가 아니다. 가격이 경쟁사의 1/10 수준이었기 때문이다.

그리고 2026년 5월 31일, DeepSeek가 단행한 결정:
> V4 Pro와 Flash의 API 가격을 75% 영구 할인했다.

| 모델 | 입력 (1M tokens) | 출력 (1M tokens) |
|------|-----------------|-----------------|
| DeepSeek V4 Pro | $0.435 | $0.87 |
| DeepSeek V4 Flash | $0.14 | $0.28 |

GPT-4o 대비 Pro는 1/12, Flash는 1/35 수준이다.

---

#### 1.3 컨텍스트 캐싱: 커피값으로 하루 종일 쓰는 비결

여기에 더해 DeepSeek는 컨텍스트 캐싱(Context Caching) 을 지원한다. 같은 시스템 프롬프트나 자주 참조하는 문서는 서버에 저장해두고, 재사용할 때는 최대 90% 할인된 가격으로 계산된다.

| 구분 | 캐싱 미적용 | 캐싱 적용 (85% 적중) | 실제 하루 비용 |
|------|-----------|-------------------|--------------|
| V4 Pro Flash | $0.87 | $0.13 | $0.13 |
| V4 Flash | $0.28 | $0.057 | $0.057 |

하루 1,000만 토큰을 처리해도 GPT-4o는 $58.75, DeepSeek V4 Flash는 $0.057.

차이가 무려 1,030배다.

#### 1.4 Mimo 2.5: 이미지 분석은 여기서

이미지 분석이나 멀티모달 작업이 필요할 때는 중국 샤오미의 Mimo 2.5를 사용한다.

| 구분 | 가격 |
|------|------|
| Mimo 2.5 입력 | $0.14 / 1M tokens |
| Mimo 2.5 출력 | $0.28 / 1M tokens |

DeepSeek Flash와 동일한 가격대지만 이미지 분석이 가능하다는 게 차이점이다.
