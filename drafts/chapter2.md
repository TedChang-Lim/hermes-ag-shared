## 2.1 컨텍스트 캐싱이란?

AI API를 쓸 때마다 매번 모든 대화 내용을 처음부터 전송하면 비용이 많이 듭니다. 대부분의 작업에서 우리는 같은 시스템 프롬프트, 같은 문서, 같은 설정 파일을 반복해서 보내게 됩니다.

**컨텍스트 캐싱(Context Caching)** 은 이렇게 자주 사용되는 내용을 서버에 저장해두고, 다음 요청 때는 "아, 이건 아까 본 거네" 하면서 **할인된 가격**으로 처리해 주는 기술입니다.

비유하자면:
> 매번 레시피를 새로 사는 대신, 한번 산 레시피는 냉장고에 두고 재료값만 내는 것과 같습니다.

## 2.2 DeepSeek의 캐싱 구조

DeepSeek V4 시리즈는 **자동 컨텍스트 캐싱**을 지원합니다. 사용자가 별도로 설정할 필요 없이, 같은 프롬프트 접두사(Prefix)가 반복되면 자동으로 캐싱이 적용됩니다.

| 항목 | 설명 |
|------|------|
| **적용 방식** | 자동 (사용자 추가 설정 불필요) |
| **할인율** | 입력 토큰 기준 **최대 90% 할인** |
| **캐싱 단위** | 프롬프트의 접두사(Prefix) 일치 시 자동 적용 |
| **세션 유지** | 단일 대화 세션 내에서 동일 컨텍스트 유지 시 효과 극대화 |

## 2.3 실제 숫자로 보는 캐싱 효과

에이전트가 매일 1,000만 토큰의 프롬프트를 전송하고, 200만 토큰을 응답받는다고 가정해 보겠습니다.

**캐싱 없을 때 (Cache Hit Rate 0%):**

| 모델 | 일일 비용 | 월 비용 |
|------|---------|---------|
| DeepSeek V4 Pro | $14.79 | $443.70 |
| DeepSeek V4 Flash | $4.36 | $130.80 |
| GPT-4o | $587.50 | $17,625.00 |

**캐싱 85% 적중 시 (실제 마스터님 환경 기준):**

| 모델 | 일일 비용 | 월 비용 |
|------|---------|---------|
| DeepSeek V4 Pro | $2.76 | $82.87 |
| DeepSeek V4 Flash | $0.89 | **$26.67** |
| GPT-4o | $58.75 | $1,762.50 |

> **DeepSeek V4 Flash + 캐싱 85% = 월 $26.67**
> 
> 같은 작업을 GPT-4o로 하면 **월 $1,762.50**

차이가 **66배**입니다.

## 2.4 캐싱 적중률을 높이는 5가지 비결

### ① 시스템 프롬프트 고정하기

에이전트에게 주는 시스템 프롬프트(행동 지침)는 매번 똑같이 유지하세요. 매번 바꾸면 캐싱이 초기화됩니다.

```yaml
# 나쁜 예: 매번 다른 프롬프트
- "너는 도움이 되는 AI 비서야"
- "너는 코딩 도우미야"

# 좋은 예: 항상 같은 프롬프트
- "You are a helpful AI coding assistant. Follow these rules: ..."
```

### ② 자주 참조하는 문서는 고정

같은 파일(.clinerules, config.yaml, README.md)을 매번 첨부하면 캐싱 효과가 누적됩니다.

### ③ 대화 세션을 길게 유지

새 세션을 시작할 때마다 캐싱이 초기화됩니다. 따라서 가능한 한 같은 세션에서 오래 작업하는 것이 유리합니다.

### ④ 불필요한 파일 읽기 반복 금지

에이전트가 같은 파일을 여러 번 읽으면 캐싱이 적용되긴 하지만, 첫 번째 읽기는 캐싱 미스(Cache Miss)로 처리됩니다. 한 번 읽은 내용을 메모리에 의존하도록 .clinerules에 지시하세요.

### ⑤ 출력 토큰도 최소화

캐싱은 입력(프롬프트)에만 적용됩니다. 출력(Completion) 토큰은 할인이 없으므로, 에이전트가 불필요하게 긴 설명을 출력하지 않도록 설정하세요.

## 2.5 실전: .clinerules 캐싱 최적화 템플릿

이 가이드의 Pro 패키지에 포함된 `.clinerules.template`을 사용하면 캐싱을 극대화할 수 있습니다:

```yaml
# 💸 Cost Optimization Guidelines
## Optimize for Cache Hits:
- 시스템 프롬프트와 지침을 일관되게 유지하세요
- 시스템 파일을 임의로 변경하지 마세요
- 큰 파일은 한 번만 읽고, 메모리나 캐싱에 의존하세요
## Minimize Token Output:
- 간결한 설명에 집중하고 긴 요약은 피하세요
- 파일 전체를 다시 쓰는 대신 특정 줄만 교체하세요
## Session Re-initialization:
- 대화 기록이 50,000 토큰을 초과하면 새 세션을 시작하세요
```

## 2.6 정리: 캐싱이 가져오는 혁명

| 캐싱 적중률 | DeepSeek V4 Flash 월 비용 | GPT-4o 월 비용 |
|:----------:|:------------------------:|:--------------:|
| 0% | $130.80 | $17,625.00 |
| 50% | $65.40 | $8,812.50 |
| 70% | $39.24 | $5,287.50 |
| **85%** | **$26.67** | **$1,762.50** |
| 95% | $13.08 | $881.25 |

캐싱 적중률을 85% 이상으로 유지하면, AI 에이전트를 24시간 가동해도 **월 3만 원 미만**으로 운영할 수 있습니다.
