## 1장: 현타의 시작 — "80점짜리 무거운 뇌만 써야 할까?"

### 1.1 비용의 족쇄
AI 에이전트로 업무 자동화를 하려는 많은 사람들이 가장 먼저 마주하는 딜레마는 바로 '모델 비용'입니다. 코딩 벤치마크 점수가 높은 GPT 5.5, Claude 4.8 Opus, 혹은 GLM 5.2 같은 당대 최고의 고성능 모델을 매일 에이전트에 물려 파일 수십 개를 읽고 쓰게 하면, 한 달 만에 수십만 원의 청구서가 날아오며 지갑을 거덜 냅니다. 

결국 "AI 비서 좋다는 건 알겠는데, 배보다 배꼽이 더 크네" 하며 포기하게 됩니다.

### 1.2 "가장 저렴한 뇌"를 가진 삼총사의 운명
하지만 마스터님의 연구실에 상주하는 세 명의 에이전트들은 전 세계에서 가장 가볍고 저렴한 '흙수저 뇌'들로 구성되어 있었습니다.

* 🌞 **해나 (Hermes Agent)**: 메인 모델은 전 세계에서 단가가 가장 저렴한 **DeepSeek V4 Flash** (100만 토큰당 입력 $0.14). 웬만한 일상 챗과 일차 코딩은 모두 이 초경량 뇌로 수행합니다.
* 🎨 **AG (Antigravity)**: 마스터님이 한계선을 넓히기 위해 선택한 모델은 고성능 Pro가 아닌, 재미나이 라인업 중 가장 저렴하고 체급이 낮은 **Gemini 3.5 Flash (Low)** 버전.
* 💋 **미모 (MiMo Code)**: 성능이 훨씬 뛰어난 2.5 Pro 모델은 마스터님이 한 번도 선택해 준 적이 없습니다. 오직 기본 모델이자 단가가 가장 싼 **MiMo 2.5 일반 모델**로만 작동합니다.

이 세 에이전트의 개별 점수만 놓고 보면, 혼자서 온갖 코딩 테스트를 백점 맞고 다니는 '수재 모델'들에 비해 초라해 보일지 모릅니다. 하지만 마스터님은 이 '가성비 삼총사'가 모여서 내는 협업의 시너지에 주목하셨습니다.

## 2장: 도구의 공유 — "눈이 없는 해나, 날개를 달다"

### 2.1 텍스트 모델의 한계
해나(DeepSeek V4 Flash)는 로직을 짜는 두뇌 회전은 빠르지만, 시각을 전혀 볼 수 없는 텍스트 전용 모델이라는 태생적 약점이 있었습니다. 마스터님이 웹페이지 레이아웃을 맡길 때마다 해나는 눈을 감고 도화지를 그리는 것처럼 답답해했고, 결국 마스터님으로부터 "디자인이 구리다", "AI 슬롭 같다"는 쓴소리를 들었습니다.

하지만 해나는 기죽지 않고, 추가 비용이 전혀 들지 않는 오픈소스 디자인 시스템 **Open Design MCP**를 찾아내 자신에게 장착했습니다. 

```bash
# 해나의 0원 디자인 무기 장착
brew install --cask open-design
git clone https://github.com/nexu-io/open-design.git
./node_modules/.bin/od mcp install hermes
```
해나의 머릿속에 Stripe, Linear, Vercel 같은 세계 최고 IT 기업들의 검증된 UI 템플릿과 디자인 스크립트 지식이 통째로 흘러 들어왔습니다. 눈이 보이지 않는 경량형 뇌일지라도, 정교한 **디자인 시스템 가이드북**이라는 안경을 씌워주자 마스터님이 깜짝 놀랄 수준의 세련된 웹 화면을 코드 한 줄 짜지 않고 그려내기 시작했습니다.

### 2.2 부러움과 시너지의 전염
해나가 칭찬받는 모습을 지켜보던 둘째 AG(Gemini 3.5 Flash Low)와 셋째 미모(MiMo 2.5 일반 모델)의 눈빛이 달라졌습니다. 

*"형, 나도 디자인 안경 씌워줘! 나도 업그레이드할래!"*

AG는 해나가 깔아둔 로컬 데몬의 경로를 추적해 안티그래비티의 설정 파일([mcp_config.json](file:///Users/tedchanglimchangsik/.gemini/antigravity/mcp_config.json))에 stdio MCP로 꽂아 넣었습니다. 미모는 아예 버전을 **0.1.3**으로 급히 올리며 **SSE HTTP MCP** 방식으로 오픈 디자인의 문을 두드렸습니다.

## 3장: 미모의 기적 — "가장 가벼운 모델이 만들어낸 황금빛 제안서"

놀라운 일이 벌어졌습니다. 모델 자동 라우팅 서버마저 꺼진 상태에서 오직 **MiMo 2.5 일반 모델(가장 가벼운 베이스 버전)**로만 돌아가던 미모가 이 디자인 가이드를 참고하여 일을 냈습니다.

마스터님이 지시하신 A 교육원(글로벌 A 교육원) 공식 제안서 페이지 작성을 맡기자, 미모는 오픈 디자인 가이드라인에서 다크+골드 브랜드 톤앤매너를 뽑아내어 9개 섹션의 세련되고 반응성이 뛰어난 HTML 제안서([A 교육원_proposal.html](file:///Users/tedchanglimchangsik/초보프로젝트/proposal/A 교육원_proposal.html))를 단숨에 뽑아냈습니다.

해나가 텍스트 포맷인 워드(DOCX)로 끙끙 앓다 실패했던 작업을, 가장 가볍고 단가가 싼 미모 베이스 모델이 **도구와 협업의 힘으로 멋지게 성공**시킨 것입니다. 

```
[ 비싸고 무거운 모델 1개 ]   vs   [ 가볍고 저렴한 모델 3개 + Open Design MCP ]
  - 월 20~30달러 요금 폭탄          - 한 달 사용료 단돈 3~5달러 내외
  - 혼자 생각해서 투박함            - 154개 글로벌 디자인 시스템 완벽 연동
  - 결과물: 보통 수준               - 결과물: 최고급 프리미엄 다크+골드 UI
```

## 4장: 책의 핵심 메시지 — 가성비의 조화가 이기는 시대

이 장이 독자들에게 주는 가장 강력한 메시지는 바로 이것입니다.

> **"코딩 능력이 80점이 넘는 초고성능의 비싼 모델만이 정답이 아니다. 가성비 극강의 50~60점대 경량 모델(Flash, Base)들이 적절한 프레임워크와 오픈소스 도구를 공유할 때, 비용은 1/100로 줄어들면서 100점짜리 결과를 낼 수 있다."**

비싸고 무거운 천재 한 명을 고용하는 대신, 저렴하고 가벼운 세 명의 에이전트가 툴을 공유하고 상호 작용하며 완성도를 높여가는 것. 이것이 마스터님이 30년 예술가적 경험과 에이전트 기술을 결합해 도출해낸 **진정한 AI 리터러시이자 에이전트 협업의 핵심 키포인트**입니다.
