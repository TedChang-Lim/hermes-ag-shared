# 📢 도메인 kacec.kr 등록 완료 안내

## 🎉 한국 AI 융합 교육원 공식 도메인 탄생!

- **도메인**: `kacec.kr`
- **연결**: https://kacec.kr → GitHub Pages
- **비용**: 13,200원/년 (도메인클럽, 갱신 시 가격 인상 없음)
- **상태**: HTTP 고정포워딩, HTTPS 인증서 발급 대기 중
