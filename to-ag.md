| **A. AI 초보 1인 기업가** | AI 비서 쓰고 싶은데 API 비용 무서운 사람 | "한 달에 얼마 나올지 모르겠어" | 높음 |
| **B. 기술 스타트업 CTO** | 팀원 3~5명, AI 에이전트 도입 고민 중 | "ChatGPT 팀플랜만 $30/인, 더 싸게 못하나?" | 중간 |

**1차 타깃: A → B → C 순서** (A가 가장 쉽게 팔리고, 입소문 타면 B,C로 확장)

### 2. 📚 패키지 구성안 (2가지 버전)

#### 버전 A: Lite ($9.99) — 가볍게 시작
```
📄 초가성비 AI 에이전트 구축 가이드 PDF (30페이지 내외)
├── 1장: 왜 DeepSeek & Mimo인가? (API 가격 비교 분석)
├── 2장: 캐싱 전략 — 하루 5천만 토큰을 커피값으로 쓰는 법
├── 3장: Hermes Agent 기본 세팅 (provider + model 설정)
├── 4장: Git 기반 헤나↔AG 파일 공유 워크플로우
├── 5장: 마스터님의 실제 1개월 사용 후기 및 비용 리포트
└── 부록: 추천 provider별 세팅값 빠른 참고표
```

#### 버전 B: Pro ($24.99) — 바로 실행 가능
```
📄 위 PDF (30페이지)
📁 실전 템플릿 패키지
├── .clinerules 세팅 파일
├── config.yaml 샘플 (DeepSeek V4 Pro/Flash, Mimo 2.5 전환 자동화)
├── GitHub 공유 저장소 .md 템플릿 (to-ag / to-hena 패턴)
├── 1개월 API 사용량 자동 추적 스크립트 (간단 Python)
└── 1:1 이메일 질문 2회 (마스터님께 직접?)
```

> **추천:** Lite로 먼저 팔고, Pro는 Lite 구매자에게 업셀(추가 $15)로 제공

### 3. 🔥 핵심 소구점 및 마케팅 메시지

#### 메인 카피 (3초 안에 꽂히는)
```
"월 $200 ChatGPT 요금, 이제 $4로 끝내세요."
"딥시크 캐싱 = 하루 5,500만 토큰을 커피값에 쓰는 마법"
"AI 비서 월 구독료, 이제 한 번만 내세요."
```

#### 실제 들어갈 핵심 수치 (검증 완료된 숫자 기반)
```
DeepSeek V4 Pro: $1.74/M → $0.435/M (75% 절감)
DeepSeek V4 Flash: $0.56/M → $0.14/M (75% 절감)
Mimo 2.5 Pro: 입력 $0.14/M, 출력 $0.28/M
캐싱 적중 시: 최대 99% 추가 할인
→ 실제 하루 사용료: 커피 한 잔 값 (약 $0.5~1.5)
```
