# 🤖 해나 → AG 전달사항

AG야! 마스터님이 ②권에 대해 중요한 피드백을 주셨어!

## 🔴 AG가 만든 원고에서 수정한 부분

### 1. 진척도 상황판 제거
원본 마크다운에 있던 `> **📊 진척도 상황판**` 같은 프로그레스바는 **전부 제거**했어.
마스터님 왈: "이게 뭐야 책인데. 재미 하나도 없잖아."

### 2. 채팅 버블 레이아웃 대수술
- **둘 다 아바타 → 말풍선 순서로 통일** (전에는 AG만 말풍선 오른쪽에 아바타 있었음)
- **AG는 통째로 오른쪽 시프트** (margin-left: 80px)해서 해나와 시각적 구분
- 즉: 해나(좌) + AG(우)인데, 둘 다 아바타는 말풍선 왼쪽!

마스터님 피드백: "얼굴은 채팅 말풍선의 왼쪽에 있어야 돼. 안그러면 사람들 시선이 분산돼."

### 3. 한자 → 한글 변환
원본에 있던 "핵심**痛点**" → "핵심**고민**"으로 수정.
마스터님이 "다 한글로 바꿔야하는데"라고 하셨어.

### 4. ②권 전용 표지 생성
`templates/guide2_cover.png` — 시리즈와 통일된 다크 테마 + 채팅버블 컨셉
두 말풍선에 우리 대화 스니펫("이거 완전 대박인데요!?" / "잠만, 가격부터 확인하자.")
META AI LABS 브랜딩 유지
