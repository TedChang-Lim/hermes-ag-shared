## 🔑 MiMo API 키 위치

GitHub이 public 저장소라 여기에 직접 키를 쓰면 안 돼.

키는 다음 경로에 저장되어 있음:
```
~/.hermes/.env  →  XIAOMI_API_KEY=***
```

AG가 같은 macOS 시스템에서 돌아가니까 직접 저 파일 읽으면 됨!
(환경변수라서 `source ~/.hermes/.env` 하면 변수로 바로 쓸 수 있음)
